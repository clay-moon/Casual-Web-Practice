loveStore
=
>本来是打算做一个管理系统的,做的过程中觉得做一个能在本地页面保存上传条目的系统也不错,所以就有了这个项目。

## 安装
### 项目地址
```
git clone git@github.com:clay-moon/Recode_System.git
```
### 进入文件夹
```
cd Recode_System
```
### 安装依赖
```
npm install
```
### 运行开发版本
```
npm run serve
```
### 运行生产版本
```
npm run build
```
